# VectorCounter
Contador animado de números por JavaScript

Instalación
---
Descargar el archivo `vCounter.js` o `vCounter.min.js`

Uso
---
```html
<script src="vCounter.min.js"></script>
```

Ejemplo
---
```html
<span class="contador">100</span>
<script>
    $('.contador').vCounter();
</script>
```
